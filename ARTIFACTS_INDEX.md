## Figures generated by other pipelines

Figures in this repository are produced by separate visualization scripts and pipeline runners, not by the test scripts above. Key figure directories:

- `figures/` -- Core analysis figures (41 PNGs including `sparc_full_results.png`, `full_pipeline_results.png`, `beat_nfw_v3.png`, etc.)
- `paper/figures/` -- Publication figures (`fig1_identity.png`, `fig2_variance.png`, `fig3_kurtosis.png`, `fig4_inversion.png`)
- `analysis/results/` -- Per-analysis figures (e.g., `injection_recovery_power.png`, `simulation_coherence_*.png`, `cluster_*.png`)
- `analysis/results/gpp_corrected/` -- GPP corrected plots (`plot_A_cs2_profiles.png` through `plot_E_subsample_comparison.png`)
- `analysis/results/residualization/` -- Residualization diagnostics (`plot_A_raw_vs_residualized_targets.png` through `plot_D_acf_lag1_env_histograms.png`)
- `analysis/results/lcdm_phase_diagram/` -- LCDM phase diagrams (`plot_a_variance_vs_gbar.png`, `plot_b_window_sweep_stability.png`, `plot_c_permutation_null.png`)
- `outputs/universality_audit/` -- Universality audit figures (10 PNGs including `three_panel_summary.png`)
- `outputs/20260223_192210/figures/` -- Timestamped run figures (`fig_alpha_star.png`, `fig_mass_matched_phase.png`, `fig_phase_null.png`, `fig_xi_organizing.png`)
- `outputs/gdagger_hunt/` -- Scale-scan and referee-proof figures across multiple timestamped runs
