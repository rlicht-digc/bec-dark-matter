Notice: Possible errata about galaxy names:

(1) File names and identification with individual galaxies
might include some confusion. If individuality of galaxies 
is necessary, please check the correspondence of the file 
names with those in the original papers cited in the paper.

This is because the paper was aimed at statistical correlations
among the fitted parameters for a large number of galaxies, but not 
at analyzing individual galaxies characteristics.

(2) Presented are Gaussian smoothed rotation curves used for
the statistics in the paper. For explanations and references, 
see the manuscript and rc_atlas_web.pdf.

(3) When using these RCs, cite the original papers listed
in the references of the paper.
