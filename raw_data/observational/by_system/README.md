# Observational Data by System

This folder provides quick-entry links for commonly referenced systems/clusters.

- `virgo/`: Virgo-targeted files and catalogs
- `coma/`: Coma-targeted files and caches
- `sagittarius/`: links to catalogs containing Sagittarius dSph records
- `fornax/`: links to catalogs containing Fornax records

Notes:
- Some source files are multi-system catalogs; links here are navigational shortcuts.
- For full source hierarchy, browse `raw_data/observational/`.
