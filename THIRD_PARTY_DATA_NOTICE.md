# Third-Party Data Notice

This repository contains third-party observational and simulation-derived datasets.

## Policy
- Third-party datasets retain their original licenses and terms.
- Reuse and redistribution follow original source terms, not this repository's code/doc licenses.
- If redistribution status is unclear, treat raw files as non-redistributable.

## Default handling used in this project
- Do not redistribute raw third-party files publicly unless explicit permission is present.
- Provide source links, required citations, and checksums so others can re-acquire data.

## Required practice for OSF uploads
- Upload project-authored code/docs under project licenses.
- For third-party data, upload only if original terms clearly allow redistribution.
- Otherwise upload metadata, manifests, checksums, and download instructions.
