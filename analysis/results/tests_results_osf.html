Supporting Test 1: test_cluster_scale_rar

Script: analysis/pipeline/test_cluster_scale_rar.py

Description: Multi-scale RAR analysis: Amram cluster galaxy rotation curves + published cluster-scale RAR results (Tian+2020, Chan & Del Popolo)

Summary Artifact: analysis/results/summary_cluster_scale_rar.json

MetricValue
level1_amram.n_galaxies8
level1_amram.n_total_points85

Finding: INCONCLUSIVE but informative. Amram cluster galaxy data lacks baryonic decomposition for direct RAR construction. Published cluster-scale RAR shows a different acceleration scale (~17× g†), which either indicates mass-dependent condensation or distinct cluster physics. This is an open question that does NOT undermine the galaxy-scale result but does constrain the theory. The Letter should note this as a testable prediction: if g† is truly universal, improved cluster baryonic mass measurements should bring a0_cluster closer to g†.
